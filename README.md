# Rooted Capital Co. | Digital Infrastructure
**Architect:** Alyssa L. (Cloud Solution Architect)

## 🎯 Mission
To eliminate operational "circus" tasks, prevent payroll fraud, and establish a high-end digital brand that rivals Sotheby’s and The Oppenheim Group.

## 🏗️ Ecosystem Architecture
- **apps/ops-portal**: Private mobile-first tool for crew time-tracking and automated payroll validation.
- **apps/luxury-site**: Public-facing editorial brand engine for HNW lead generation (splitspace.co).
- **packages/logic**: Centralized business rules (Rates: $50 Foreman / $25 Laborer).

## 🔒 Security & Governance
- **Private Repository**: All proprietary labor data and investor logic is stored in this secure vault.
- **Data Integrity**: Hardcoded rate validation to prevent manual "math errors" and overbilling.
